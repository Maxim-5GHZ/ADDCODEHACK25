vari_map - индекс VARI - Видимый атмосферно-устойчивый индекс
ndvi_map - индекс NDVI - Нормализованный дифференциальный вегетационный индекс
rgb_path - путь до rgb картинки
nir_path - путь до nir картинки
image_provider - картинка,получаемая через API
<Class>.from_gee - метод получения API (аргументы lat - широта,lon - долгота,start_date - дата начала,end_date - дата конца)
image_provider.rgb_image - rgb спутниковое изображение
image_provider.nir_channel - nir спутниковое изображение
image_provider.red_channel - red спутниковое изображение
mean_value - среднее значение плотности зелёной биомассы - чем больше, тем плотнее раститеьность
std_dev_value - стандартное отклонение - чем больше отклонение,тем неоднороднее растительный покров
percentages - "Почва/Низкая растительность (<= 0.2)" "Средняя плотность (0.2 - 0.5)" "Высокая плотность (0.5 - 0.8)" "Очень высокая плотность (> 0.8)"
