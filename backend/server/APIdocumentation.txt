Имя запроса и его описание; Запрос внутри fetch
Получение токена авторизации; fetch(`/get_token?login=${encodeURIComponent(login)}&password=${encodeURIComponent(password)}`)
Регистрация нового пользователя; fetch(`/add_user?login=${encodeURIComponent(login)}&password=${encodeURIComponent(password)}`, {method: 'POST'})
Сохранение данных пользователя; fetch(`/savedata?token=${encodeURIComponent(token)}&key_array=${encodeURIComponent(keyArray)}`, {method: 'POST'})
Получение данных пользователя; fetch(`/givefield?token=${encodeURIComponent(token)}`)
Получение логов администратора; fetch(`/log?password=${encodeURIComponent(password)}`)
Проверка здоровья сервера; fetch('/health')
Получение главной страницы; fetch('/')
Получение favicon; fetch('/favicon.ico')
Обновление данных пользователя; fetch(`/data/update?token=${token}&key_array=${keyArray}`, {method: 'PUT'})
Частичное редактирование данных; fetch(`/data/edit?token=${token}&new_keys=${newKeys}&keys_to_add=${keysToAdd}&keys_to_remove=${keysToRemove}`, {method: 'PATCH'})
Удаление данных пользователя; fetch(`/data/delete?token=${token}`, {method: 'DELETE'})
Проверка существования данных; fetch(`/data/check?token=${token}`)
Установка данных поля; fetch(`/field/set?field=${encodeURIComponent(field)}&data=${encodeURIComponent(data)}&token=${encodeURIComponent(token)}`, {method: 'POST'})
Получение данных поля; fetch(`/field/get?field=${encodeURIComponent(field)}&token=${encodeURIComponent(token)}`)
Удаление данных поля; fetch(`/field/delete?field=${encodeURIComponent(field)}&token=${encodeURIComponent(token)}`, {method: 'DELETE'})
Проверка существования поля; fetch(`/field/check?field=${encodeURIComponent(field)}&token=${encodeURIComponent(token)}`)
